# 개발 유용한 기능

- `shift` + `shift` : 전체 검색기능 
- `alt` + `7` : 클래스 구조 보기
- `Shift` + `option` + `화살표 위/아래` : 현재 라인 옮김
- `Shift` + `command` + `화살표 위/아래` : 현재 블럭 옮김
