+,-교행 분수 수열: 분수 수열에 부호(+,-)만 교대로 바껴서, 부호가 교행하는 수열

<br>


## 설명
---

- +,-교행 분수 수열
	+ 분수 수열에 부호(+,-)만 교대로 바껴서, `부호가 교행하는 수열`
	1. 부호 +,-를 한번에 처리 방식
	2. 스위치 변수 사용하는 방식
	3. 홀/짝 검사해서 처리 방식

<br>

$ C = \frac{A}{B} $

$$ C = \frac{A}{2X3} $$

$$ S = \frac{1}{2X3} - \frac{2}{3X4} + \frac{3}{4X5} $$




## 문제
---

- $$ S = \frac{1}{2X3} - \frac{2}{3X4} + \frac{3}{4X5} $$
- $$ C = \frac{A}{2X3} $$
- 알고리즘을 제시하라

<br>


> C언어

```c
#include<stdio.h>

int main() {

   	int N = 0;
	int S = 0;
	int SW = 0; // 교행 부호를 위한 스위치 변수
	do {
	    N++;
	    if (SW == 0) {
	        S += N;
	        SW = 1;
	    } else {
	        S -= N;
	        SW = 0;
	    }
	} while (N <= 10);
	printf("%d\n", S);
};
```
