IntelliJ Lombok 적용


<br><br>

# 1. IntelliJ Lombok 플러그인 설치

1. File > Setting
2. 왼쪽 메뉴에서 plugin 선택
3. lombok 설치
4. restart

<br>


# 2. Enable annotation 설정

1. File > Setting (Ctrl+Alt+S)
2. Build, Execution, Deployment > Compiler > `Annotation Processings`
3. Enable annotation processing 체크
